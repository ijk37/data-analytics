## What You Will Learn

- Describe data correctly using measurement scales, distributions, and robust summary statistics.
- Investigate relationships with covariance, correlation, multivariate summaries, and visual diagnostics.
- Prepare trustworthy datasets by handling missing values, outliers, encoding, transformation, and scaling.
- Discover structure with k-means, hierarchical clustering, DBSCAN, and distance measures.
- Find co-occurrence patterns with Apriori, FP-growth, support, confidence, and lift.
- Build and evaluate classifiers with decision trees, k-NN, Naive Bayes, cross-validation, and error metrics.

## Course Flow

1. **Understand:** read the chapter notes and reproduce the key ideas in your own words.
2. **Practice:** solve the matching exercise set before opening the worked answers.
3. **Recall:** take the randomized chapter quiz and review every explanation.
4. **Apply:** run a matching project, modify one assumption, and explain the result.

```text
Notes → Exercises → Quiz → Project → Review weak areas
```

## Choose a Learning Route

### New to Analytics

Follow Modules 01–08 in order. For each module, read the notes, answer the exercises without peeking, take the quiz, and run one project.

[Begin with Module 01](01-notes/01-data-analytics-introduction.md)

### Reviewing for an Assessment

Start with the [ten-minute course summary](01-notes/08-summary.md), identify weak topics with the decision guide, then complete the [mixed review](02-exercises/08-exercise.md) and randomized quizzes.

### Building a Portfolio

Complete one focused lab from Modules 02, 04, 05, 06, and 07, then turn a [Module 08 capstone](04-projects/README.md#module-08-final-projects-capstones) into a documented case study.

## Capstone Practice

Seven end-to-end capstones connect data auditing, exploratory analysis, preparation, pattern discovery, modeling, evaluation, and interpretation. Scenarios include student performance, retail behavior, medical risk, market baskets, and the Iris dataset.

[Explore all projects](04-projects/README.md)

## Getting Started

You do not need to install anything to read, practice, or take a quiz. When you are ready to run the code, use the [learning toolkit](05-resources/README.md) for setup guidance, dataset preparation, and a project write-up checklist.
