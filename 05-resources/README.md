This folder holds the original course material the notes, exercises, and projects were built from. It stays **local-only** — the `Class Lectures/` and `Class Exercises/` subfolders are excluded from git via `05-resources/.gitignore` because they contain copyrighted lecture slides and assignment files (PDF/DOCX/PPTX/XLSX).

## &#128218; What's here (locally)

| Folder | Contents |
| --- | --- |
| `Class Lectures/` | Chapter lecture slides (`ch01.pdf` … `ch07.pdf`) with extracted text companions |
| `Class Exercises/` | Solved assignment sources (`01-excercise` … `08-excercise`) in DOCX/PDF/XLSX, used to build the published [Exercises](../02-exercises/README.md) |

## &#128161; Published Version

The worked-through, cleaned-up version of this material is published as:

- **[Notes](../01-notes/README.md)** — the concepts from each lecture, rewritten as structured study notes.
- **[Exercises](../02-exercises/README.md)** — the solved assignments above, reformatted into the site's practice-question style.

If you're browsing this repo on GitHub, only this README is visible for this folder — the source files are intentionally not pushed.
